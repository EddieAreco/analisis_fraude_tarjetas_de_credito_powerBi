# analisis_fraude_tarjetas_de_credito_powerBi
Proyecto de análisis de fraude desarrollado de punta a punta en Power BI, incluyendo limpieza de datos, medidas DAX (donde se usaron funciones condicionales como IF y SWITCH), definición de KPIs y dashboards interactivos para analizar patrones de fraude, niveles de riesgo y montos involucrados
